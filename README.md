# 💵 Cash Calculator

A clean and responsive **currency conversion** web app built with **React** and **Next.js**. Instantly convert amounts between world currencies with a simple and intuitive interface.

---

## 🚀 Features

- 🔄 Convert between multiple world currencies
- ⚡ Fast and responsive UI built with React
- 🌐 Server-side rendering powered by Next.js
- 📱 Mobile-friendly design
- 🎯 Clean and minimal interface

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| [React](https://react.dev/) | UI components & state management |
| [Next.js](https://nextjs.org/) | Framework & routing |
| CSS / Tailwind | Styling |

## 📦 Getting Started

### Prerequisites

- Node.js `v18+`
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/cash-calculator.git
   cd cash-calculator
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```
